# 簡易文字冒險遊戲 (Bash Adventure Game)

這是一個用 **Bash Shell Script** 編寫的文字冒險遊戲。  
玩家可以探索森林、對抗怪物、收集道具，最終打敗守護者成為勇者。

---

## 🖥️ 系統需求
- 建議環境：Linux / macOS  
- Windows 使用者可透過以下方式執行：
  - **WSL (Windows Subsystem for Linux)**
  - **Git Bash**
  - 或其他能執行 Bash 的終端機

---
